# Pokémon Pocket Battle ⚡

A fast-paced, streamlined Pokémon TCG experience designed for quick strategic sessions.

## 🎮 Game Overview
**Pokémon Pocket Battle** reimagines the classic card game with a focus on "pocket-sized" matches. By using smaller decks and a simplified point system, we've accelerated the gameplay without losing the strategic depth.

### Core Mechanics
* **Compact Decks:** Players build a **20-card deck** for high-intensity play.
* **Elite Slots:** Decks are limited to **4 "Power" cards** (VMAX, ex, Mega ex, or V).
* **Energy Flow:** To keep the game moving, players gain **one energy** automatically every turn.
* **Turn Logic:** A coin flip determines the starting player.
* **Evolution:** Full support for evolving cards during the match.

### 🏆 Victory Conditions
The game uses a dynamic point system. The first player to reach **4 points** wins:
| Knockout Type | Points Awarded |
| :--- | :--- |
| **Basic Card** | 1 Point |
| **ex, VMAX, or V Card** | 2 Points |

We are currently iterating on the **Start Screen** layout. Based on our wireframes, the interface includes:

* **View Profile:** Check player statistics and history.
